# MiniCrew 개발 기획서 (Development Plan)

## 1. 프로젝트 개요
- **프로젝트명**: MiniCrew
- **목표**: 그룹(회사) 기반의 간편한 워크플로우 관리 웹 애플리케이션 구축
- **핵심 가치**: 필요한 기능만 담은 미니멀리즘, 직관적인 UI, 빠른 업무 흐름

## 2. 핵심 기능 명세
### 1) 사용자 및 그룹 (User & Group)
- **그룹**: 회사가 루트가 되며, 모든 프로젝트와 유저는 그룹에 속함.
- **유저**: 사원 개념. 로그인하여 프로젝트에 참여.

### 2) 레이아웃 (Layout)
- **헤더 (Header)**: 검색창, 알림, 우측 채팅 토글.
- **왼쪽 네비게이션 (Left Navbar)**:
  - 스크롤 가능.
  - 프로젝트 생성 버튼.
  - 내 프로젝트 목록 (즐겨찾기 포함).
  - 대시보드 링크.
  - 접기/펼치기 가능.
- **오른쪽 네비게이션 (Right Navbar)**:
  - 채팅 내역 리스트.
  - 헤더의 채팅 아이콘 클릭 시 토글.
- **메인 컨텐츠 (Main)**:
  - 양쪽 사이드바 상태에 따라 가변적인 너비.

### 3) 대시보드 및 프로젝트 (Dashboard & Project)
- **대시보드**: 참여 중인 프로젝트 요약.
- **프로젝트 상세**:
  - **탭(Category)**: 기획, 마케팅, 디자인 등 업무 분야별 탭 구성 (사용자 커스텀 가능).
  - **게시글(Post)**: 리스트가 아닌 **카드 형태**의 요약 뷰.

### 4) 게시글 (Post Card)
- **카드 구조**:
  1. **상단**: 작성자 프로필 (이름, 팀명).
  2. **제목**: 업무 제목.
  3. **진행 상태**: 버튼으로 변경 가능 (예정, 진행중, 완료, 보류, 이슈).
  4. **내용**: 업무 상세 내용.
  5. **하단**: 댓글 미리보기/입력.
- **알림**: 상태 변경 및 댓글 등록 시 알림 발생.

## 3. 개발 단계별 작업 순서 (Workflow)

### 1단계: 프로젝트 환경 설정 (완료)
- [x] Next.js 14 (App Router) + Tailwind CSS 설정
- [x] Supabase 연동 설정 (기본 클라이언트)
- [x] 아이콘 및 필수 라이브러리 설치

### 2단계: 데이터베이스 설계 (ERD)
- [ ] Users, Projects, Categories, Posts, Comments 테이블 설계.
- [ ] Supabase에 테이블 생성 및 RLS(보안) 정책 설정.

### 3단계: UI 레이아웃 구현 (퍼블리싱)
- [ ] **RootLayout**: Header + Left Sidebar + Right Sidebar + Main Content Grid 구조 잡기.
- [ ] **Sidebar Components**: 프로젝트 리스트, 채팅 리스트 더미 데이터로 구현.
- [ ] **Interaction**: 사이드바 토글 기능 및 반응형 동작 구현.

### 4단계: 핵심 기능 개발 (Front-End & API)
- [ ] **인증 (Auth)**: 로그인 페이지 및 세션 관리.
- [ ] **프로젝트**:
  - 프로젝트 생성 모달.
  - 프로젝트 리스트 조회.
  - 탭(카테고리) 생성 및 전환 기능.
- [ ] **게시글 (Post)**:
  - 카드 UI 컴포넌트 제작 (SVG 아이콘 사용, 애니메이션 제약 준수).
  - 상태 변경 로직 작성.
- [ ] **댓글 (Comment)**: 게시글 하단 댓글 작성 및 조회.

### 5단계: 고도화 및 연동
- [ ] **알림 시스템**: DB 트리거 또는 Supabase Realtime을 이용한 알림.
- [ ] **채팅**: 실시간 채팅 기능 (Supabase Realtime).
- [ ] **검색**: 프로젝트 및 게시글 검색.

## 4. 디자인 가이드라인 (Design Rues)
- **아이콘**: 반드시 **SVG**만 사용 (이미지/폰트 아이콘 금지).
- **애니메이션**: 카드 호버 시 위아래 움직임 **절대 금지**. 단순 색상 변화 정도만 허용.
- **스타일**: 깔끔한 비즈니스 대시보드 스타일 (참고 이미지 기반).
- **색상**: AI스러운 복잡한 그라데이션 금지. 솔리드한 컬러 위주 사용.
